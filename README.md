## iShiWuPai
iShiWuPai是基于React Native和Redux实现的美食类APP，接口来自《食物派》。

## 完成功能
目前只针对iOS端，完成功能：
>* 逛吃页面展示，支持下拉刷新和上拖加载更多
>* 资讯详情页面展示
>* 食物百科页面展示
>* 食物列表页面展示，营养素排序功能及动画，支持上拖加载更多
